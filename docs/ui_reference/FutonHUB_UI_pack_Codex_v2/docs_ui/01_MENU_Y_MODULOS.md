# FutonHUB UI - Menú oficial y módulos

## Menú lateral oficial del ERP

El menú principal debe mantener estos apartados visibles:

1. Dashboard
2. Inventario
3. Cambio de Precios
4. Pedidos
5. WooCommerce
6. Informes / Exportaciones
7. Seguridad / Logs
8. Configuración

## Módulos fuera de alcance por ahora

### Proveedores

El módulo **Proveedores** queda fuera de momento.

Motivo:
- No tenemos suficiente información útil de proveedores para justificar una ventana propia.
- La selección de proveedor ya se usa dentro de Pedidos como punto de entrada para calcular pedidos.
- No crear una sección grande de Proveedores por ahora.

Regla:
- No añadir “Proveedores” al menú principal.
- No diseñar ventana independiente de proveedores salvo que se indique más adelante.
- En Pedidos sí pueden aparecer tarjetas de proveedores para iniciar “Calcular nuevo pedido”.

## Nomenclatura visible

Mantener estos nombres exactos:

- Dashboard
- Inventario
- Cambio de Precios
- Pedidos
- WooCommerce

```text
Dashboard

Cambio de Precios
Pedidos
WooCommerce
Informes / Exportaciones
Seguridad / Logs

```

## Estilo del menú

- Menú lateral.
- Limpio.
- Sin textos largos.
- Icono + nombre.
- Estado activo claro.
- Mantener colores suaves.
- No saturar con submenús visibles si no son necesarios.
