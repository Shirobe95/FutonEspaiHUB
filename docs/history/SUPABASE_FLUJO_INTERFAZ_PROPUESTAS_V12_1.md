# FutonEspaiHUB v12.1 - Flujo interfaz propuestas

Esta versión cierra el circuito operativo desde la interfaz del HUB:

- Buscar producto/variación en Supabase desde una ventana visual.
- Crear propuesta desde el item seleccionado, sin copiar IDs de memoria.
- Previsualizar la propuesta antes de guardarla.
- Previsualizar una propuesta guardada antes de aprobar/rechazar.
- Aprobar/rechazar sigue siendo operación permitida para admin y worker.
- Publicar WooCommerce sigue siendo solo admin y mantiene preview + confirmación PUBLICAR.

WooCommerce solo se toca si el admin usa Publicar Woo y confirma explícitamente.
