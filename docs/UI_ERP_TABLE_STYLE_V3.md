# UI ERP - Ajuste visual de tablas v3

## Objetivo

Mejorar la lectura visual de todas las tablas del prototipo ERP.

## Cambios

- Texto de columnas centrado en `ttk.Treeview`.
- Cabeceras centradas.
- Borde ligero en Treeview y cabeceras.
- Altura de fila aumentada para respirar mejor.
- Se mantiene scroll horizontal donde las tablas son grandes.

## Regla de diseño

Las tablas deben verse ordenadas y compactas, con cada dato en su columna correcta.
No se cambia la lógica ni el origen de los datos.

## Módulos afectados visualmente

- Inventario
- Propuestas
- Pedidos
- WooCommerce
- Informes / Exportaciones
- Seguridad / Logs
- Tablas auxiliares del ERP
